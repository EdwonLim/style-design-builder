# Radio 单选框

<span class="tag ripple">Ripple</span>

选择一组选项的 **单个** 选项。

与移动端不同的是，PC 端的 Radio 组件图标要小一些，并且字号小一些(与 PC 端正文字号相同)。这是因为 Radio 在移动端是主要操作，要更明显，因此使用标题字号；而在 PC 端 Radio 组件一般是处于正文的一部分。
