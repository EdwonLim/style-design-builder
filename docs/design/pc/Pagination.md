# Pagination 分页

分页按钮（仅用于 PC 端）。

## 组件尺寸

分为两种尺寸：“大尺寸”、“小尺寸”。

大尺寸的组件高度为 40px，小尺寸的组件高度为 32px，组件的其他大小不变。
