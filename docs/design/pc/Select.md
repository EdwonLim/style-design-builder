# Select 下拉选框

Select 下拉选框仅用于 PC 端。

### 组件尺寸

分为两种尺寸：“大尺寸”、“小尺寸”。

大尺寸的组件高度为 40px，小尺寸的组件高度为 32px，组件的其他大小不变。

### 组件类型及状态

组件分为两种类型：“带背景的下拉选框” 和 “透明背景的下拉选框”。
