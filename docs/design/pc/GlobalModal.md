# GlobalModal 全局弹层

全局弹层，用于复杂一些的表单操作，一般是针对当前页面的附属表单。全局弹层覆盖在页面上方，可以具有多页的层级关系，点击左上角的按钮退出当前页，点击右上角的按钮关闭整个弹层。

该组件入侵性较强，会阻断用户的使用流程，因此非必要情况无需使用。
