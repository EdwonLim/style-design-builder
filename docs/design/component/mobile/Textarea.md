# Textarea 多行文本输入框

Textarea 用于输入多行文本。

## 分类

<div class="imgblock">
  <div class="sm">
    <p>Textarea 有不同的输入状态，如右图所示：</p>
    <p>未输入的时候显示 Placeholder，计数器是灰色，斜杠前面的数字为当前输入的字数。</p>
    <p>输入的时候，当前输入的字数显示为深色。</p>
    <p>输入的字数超过了预期的字数，当前输入的字数显示为红色(警告色)，但输入框中文本颜色不变，且可以超出字数继续输入，但不允许提交表单。</p>
